Shikoba v4.0

Software necesario para la instalación:

-php
-Apache
-mysql

Pasos para la instalación:

-Importar la base de datos desde el archivo convivencia2.

-Activar los eventos.

-Para la api, la urls las tenemos alojados en un archivo con variables globales de la aplicación pwa en el que se pueden cambiar.

-Para la aplicación del backend (symfony), con la que puedes crear las noticias y publicarlas, la lanzas con el comando php bin/console server:run o php bin/console server:start

-Inicias sesion con el usuario y contraseña convivencia.

-Para la aplicacion del frontend, se lanza desde la carpeta de producción dist/pwa con el comando angular-http-server.

Todos los comando deben ser lanzados como administrador para su correcta ejecución.
